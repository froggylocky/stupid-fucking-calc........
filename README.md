# Golfed Branding Calculator

An extremely minified ("code-golfed") scientific calculator built with HTML, CSS, and Vanilla JS.

## Features
- **Code-golfed Core**: The entire logic is compressed for minimum footprint.
- **Dynamic UI**: Features a moving checkerboard background and glowing interfaces.
- **Scientific Functions**: Supports trigonometry, logarithms, factorials, and solving for `x`.
- **Branding**: Froglock Inc. integrated assets and custom fonts.
- **Vercel Ready**: Optimized for rapid deployment.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)
- [Vercel](https://vercel.com/)
